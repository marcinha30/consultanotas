# Consulta NF-e / NFC-e

Sistema web para consulta e identificação de documentos fiscais eletrônicos a partir da chave de acesso.

## Funcionalidades

* Consulta de NF-e (modelo 55)
* Consulta de NFC-e (modelo 65)
* Identificação automática do estado emissor pela chave de acesso
* Redirecionamento para o portal oficial da SEFAZ correspondente
* Cópia rápida da chave de acesso
* Interface responsiva para desktop e dispositivos móveis
* Armazenamento da última chave consultada

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript Vanilla

## Como Utilizar

1. Digite ou cole a chave de acesso com 44 dígitos.
2. Clique em **Consultar**.
3. O sistema identificará o estado emissor e o tipo de documento.
4. Clique no botão de consulta para acessar o portal oficial da SEFAZ.

## Objetivo

Este projeto foi desenvolvido para facilitar a consulta de notas fiscais eletrônicas (NF-e e NFC-e) através dos portais oficiais das Secretarias de Fazenda dos estados brasileiros.
